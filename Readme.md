# 🎨 Arjun A Acharry – Interactive Portfolio Website

A modern, interactive, Figma-inspired portfolio website showcasing the professional journey, projects, achievements, and design philosophy of **Arjun A Acharry**, a UI/UX Designer and Creative Professional from Kerala, India.

## ✨ Overview

This portfolio is designed to mimic the experience of navigating a collaborative design file, inspired by modern design tools such as Figma. It combines storytelling, interaction, and visual design to create an engaging experience for visitors while presenting professional achievements and work.

### Live Experience Includes

* Interactive Figma-style interface
* Dynamic navigation system
* Light & Dark mode support
* Animated splash screen
* Comment pins and annotation system
* Interactive drawing board
* Theme customization panel
* Responsive design for desktop and mobile
* Smooth animations and micro-interactions
* Portfolio sections organized as design artboards
